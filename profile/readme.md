# 🌈 bunq • Bank of The Free

Welcome to the official GitHub profile of [bunq](https://www.bunq.com) — the only bank built by fellow developers, for developers. We're on a mission to revolutionize banking by making it fully programmable, transparent, and instant.

## 🚀 What We Do

At bunq, we believe banking should work for you — not the other way around. That's why we’ve built a modern API-first bank from the ground up. Whether you want to automate your finances, build your own financial tools, or integrate bunq into your app, we give you full control.

## 🧠 Highlights

- **API Documentation** – All that you need to know to start building with our powerful [bunq API](https://doc.bunq.com).
- **SDKs** – Official SDKs for Python, PHP, Java, and C# to help you get started fast.
- **Webhooks** – Get real-time updates for events like payments and card transactions.
- **OAuth** – Secure third-party integrations for building financial apps with bunq.
- **Sandbox Environment** – Test everything safely with our fully-featured sandbox.

## 🧪 Get Started

1. Sign up for a free [sandbox developer account](https://developer.bunq.com)
2. Grab your API key from the [developer portal](https://developer.bunq.com)
3. Choose your favorite SDK or use direct HTTPS requests
4. Start building 🚀
